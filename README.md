# Modular-coding-
# Todo App :)))

A modern, modular JavaScript todo application built with ES6 modules.

## Live Demo
[View Live Site](https://chash-code.github.io/Modular-coding-/)

## Features
- User signup and login with localStorage
- Fetch todos from JSONPlaceholder API
- Add, edit, delete, and toggle todos
- Modular code structure with ES6 import/export
- Responsive design

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6 Modules)
- localStorage API
- Fetch API

## How to Run Locally
1. Clone this repository
2. Open `index.html` in a modern browser
3. Create an account and start managing todos!

## Project Structure
- `navbar.js` - Navigation bar component
- `footer.js` - Footer component
- `displayTodos.js` - Todo display logic
- `index.html` - Home page
- `signup.html` - User registration
- `login.html` - User authentication
- `todos.html` - Todo management page
